# 岁月史书 (Book of Ages)

> 个人数字剪报本 — 收集值得记录的事件，梳理发展脉络，关联参考材料。

## 这是什么

想象一本笔记本，你每天把看到的新闻、事件、思考剪贴进去，配上时间线和附件。这个系统就是它的数字化版本，专门为你和 AI Agent 协作而设计。

## 核心能力

| 能力        | 说明                                               |
| ----------- | -------------------------------------------------- |
| **📥 收集** | 手动录入 / URL 一键剪藏 / Agent 通过 API 推送草稿  |
| **📋 审批** | 草稿进入收件箱，你浏览后一键收录或忽略             |
| **📝 整理** | 为事件添加时间线、标签、参考材料（图片/文档/快照） |
| **🔍 检索** | 全文搜索、标签筛选、时间线浏览                     |

## 技术栈

- **前端**: Vue 3 + Naive UI + TailwindCSS
- **后端**: Fastify (TypeScript)
- **存储**: SQLite（单文件，零外部依赖）

## 快速开始

### 环境要求

- Node.js >= 20

### 本地开发

```bash
npm install

# 启动后端（3000 端口）+ 前端（5173 端口）
npm run dev:server & npm run dev:web
```

浏览器访问 `http://localhost:5173`。

### Docker 部署

```bash
docker-compose up -d
```

数据持久化在 `./data` 目录，适合 NAS 部署。
