# 岁月史书 (Book of Ages)

> 个人数字剪报本 — 收集值得记录的事件，梳理发展脉络，关联参考材料。

## 这是什么

想象一本笔记本，你每天把看到的新闻、事件、思考剪贴进去，配上时间线和附件。这个系统就是它的数字化版本，专门为你和 AI Agent 协作而设计。

## 核心能力

| 能力        | 说明                                              |
| ----------- | ------------------------------------------------- |
| **📥 收集** | 手动录入 / URL 一键剪藏 / Agent 通过 API 推送草稿 |
| **📋 审批** | 草稿进入收件箱，你浏览后一键收录或忽略            |
| **📝 整理** | 为事件添加时间线、标签（支持即时创建）、参考材料  |
| **🔍 检索** | 全文搜索、标签筛选、时间线浏览                    |

## 快速开始

### 1. 运行项目

```bash
npm install

# 启动后端 (3000) + 前端 (5173)
npm run dev:server
npm run dev:web
```

浏览器访问 `http://localhost:5173`。

### 2. Docker 部署

```bash
docker-compose up -d
```

数据持久化在 `./data` 目录。
